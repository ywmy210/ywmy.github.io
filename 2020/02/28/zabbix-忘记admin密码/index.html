zabbix忘记admin密码

好长时间 没有使用admin登录zabbix了，尝试了几次没登录上去，虽然有其它用户 暂不影响使用，但问题一直在，也不是办法。

解决方法

1.连接zabbix数据库

2.查询user表信息

3.生成新的密码的md5信息

4.update表数据，userid=1的这个用户

5.刷新重新加载权限表
